# WHITE-BOX
・リセット方法 

1：nuke_universal.uf2(26/06/21時点)をダウンロード

  https://www.raspberrypi.com/documentation/microcontrollers/pico-series.html
  
2：CircuitPythonファームウェアをダウンロード

  https://circuitpython.org/board/waveshare_rp2040_zero/
  
3:BOOTボタンを押下しながらusbで接続する

4:ドライブとして認識されたら、1で入手したファイルを書き込む

5:再度立ち上がったら、2で入手したファイルを書き込む

6:本リポジトリのファイルを書き込む
